stratum

single player chess (against the engine) with a small twist!?

each separate level challenges you against an increasingly difficult starting position. each level contains three difficulties:

  • ★ (Intermediate): Roughly 1320 elo
  • ★★ (Advanced): Roughly 1800 elo.
  • ★★★ (Expert): Roughly 2200 elo.

either draw or win the game to succeed!

seems difficult? keep in mind that there are difficulty modifiers implemented in game

Use the side console to toggle game modifiers before clicking into the match

challenges listed under each level require the specific difficulty and modifiers active, in any run

Meet challenge requirements and pass the level to earn the challenge star
